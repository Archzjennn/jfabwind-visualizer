# JFABWIND Visualizer 🚀

JFABWIND Visualizer adalah aplikasi berbasis web interaktif yang dirancang untuk melakukan visualisasi data struktur chart/graph secara dinamis dan estetik. Aplikasi ini mengutamakan performa tinggi, antarmuka modern dengan dukungan tema gelap (*dark mode*), serta navigasi yang responsif.

## ✨ Fitur Utama
- **Interactive Graph/Flow Layout:** Menggunakan *layouting engine* otomatis untuk menyusun simpul (*nodes*) secara rapi.
- **Dynamic Theme System:** Perpindahan instan antara *Light Mode* dan *Dark Mode* (Deep Midnight Blue `#080810`).
- **Guided User Experience:** Fitur *onboarding tour* interaktif untuk memandu pengguna baru.
- **Export Capabilities:** Kemampuan mengunduh hasil visualisasi langsung menjadi format gambar (PNG) maupun dokumen (PDF).
- **Smooth Animations:** Efek transisi interaktif bertenaga tinggi di setiap elemen modal dan tata letak.

## 🛠️ Tech Stack
Aplikasi ini dibangun menggunakan ekosistem teknologi modern:
- **Core:** React 19 & TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS & Lucide React (Icons)
- **State Management:** Zustand
- **Visualization:** React Flow & Dagre Layout Engine
- **Animation:** Framer Motion
- **Utilities:** Driver.js, html-to-image, jsPDF
